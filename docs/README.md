\# ESP32-C3 Rotary Encoder + OLED Project



\## Overview

This project uses ESP32-C3 with a rotary encoder and SSD1306 OLED display to navigate menus and edit values.



\## Features

\- Rotary encoder navigation

\- OLED menu system

\- Editable parameters (Temp, Humidity, Voltage)

\- MicroPython firmware



\## Hardware Used

\- ESP32-C3

\- Rotary Encoder (CLK, DT, SW)

\- SSD1306 OLED (I2C)



\## Pin Mapping

\- CLK → GPIO 2

\- DT → GPIO 3

\- SW → GPIO 4

\- SDA → GPIO 8

\- SCL → GPIO 9



\## Project Structure

\- src/ → firmware code

\- hardware/ → KiCad schematics

\- docs/ → documentation
